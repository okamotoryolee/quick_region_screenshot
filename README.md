# QuickShot

QuickShot is a **lightweight screenshot utility for Windows**.  
With a simple hotkey, you can select any region on your screen,  
save it instantly to a folder, and copy it to the clipboard for immediate pasting.

## ✨ Features
- **Hotkey-driven region capture**
  - `Ctrl + Shift + A` : Select a region and capture
  - `Ctrl + Shift + Q` : Quit the app
- **Auto-organized storage**
  - Saves under `Pictures\QuickShots\YYYY-MM-DD\`
- **Clipboard integration**
  - Captured images are automatically copied to the clipboard
